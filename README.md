# Todo cli

A simple command-line task manager written in Python

## Features

- Add new tasks

- View all tasks

- Delete any tasks

- Save tasks automatically to a file

## Requirements

Python 3.12+

## Commands

'add buy bread' - add a new task

'show' - show all tasks

'delete <number>' - delete task by it's number

'exit' - exiting the program

## Installation code

```bash
git clone https://github.com/dil2324/Todo-CLI.git

cd todo_cli
```

## How to run

```bash
python todo_cli.py
```
